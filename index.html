FORUM•enhanced



FORUM • enhanced is a collaborative project that introduces a new type of forum which abstains from using the standard linear forum format. It incorporates a new search method which allows users to adjust the relevancy of the search result based on personalized tag choices, search results, posts and comments are displayed with d3 Javascript interactive bubble graph.

Technologies used: HTML5, CSS3, Ruby on Rails, D3.js, Javascript, jQuery, PostgreSQL, Bootstrap and deployed via Heroku.

  • General Assembly: Launch Project
  • Course: Web Development Immersive
  • Date: Feb 2015

NESTED



Nested is a collaborative project created to improve the roommate search tool available on the web today. Our goal is to build an open and honest rental community that helps renters find their ideal roommate to share a home with.

Technologies used: HTML5, CSS3, Ruby on Rails, MongoDB, Javascript, jQuery, Bootstrap, GoogleMaps API and deployed via Heroku.

Features currently in progress:

  • Improved Messaging feature.
  • Improved individual listing page.
  • Improved UX/UI design.

    • General Assembly: Launch Project
    • Course: Web Development Immversive
    • Date: April 2015
